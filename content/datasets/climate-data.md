Comprehensive climate data including temperature, precipitation, and weather patterns for Kurdistan Region collected from 2023 to 2024.

## Data Description

This dataset contains comprehensive climate data for the Kurdistan Region, including:

- Temperature records (daily, monthly, yearly)
- Precipitation data
- Weather patterns
- Climate indicators

## Data Format

- CSV files for analysis
- Excel files for visualization
- Metadata documentation

This dataset is available as open access. You can download it directly from the link below.

## Usage

This data has been used in several research projects and publications related to climate change impacts.
