## Laboratory Overview

The Data Analysis Laboratory provides computational resources and software tools for statistical analysis, data visualization, and computational research. The facility supports researchers from all disciplines requiring data analysis capabilities.

## Software Available

- SPSS Statistics
- R/RStudio
- Python (Anaconda)
- MATLAB
- Stata
- ArcGIS

## Research Areas

- Statistical analysis
- Data visualization
- Machine learning
- Computational modeling
- Geographic information systems (GIS)

## Access and Booking

Workstations are available on a first-come, first-served basis during operating hours. For dedicated high-performance computing resources, please contact the lab coordinator.
